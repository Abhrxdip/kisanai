# Kisan AI 🌾

An intelligent agricultural assistance platform built with React and TypeScript, designed to help farmers make data-driven decisions.

## Features

- 🌱 **Crop Recommender**: Get intelligent crop recommendations based on soil and environmental conditions
- 💧 **Fertilizer Prediction**: Optimize fertilizer usage with AI-powered predictions
- 📊 **Soil Reports**: Comprehensive soil analysis and reporting
- 🌤️ **Weather Integration**: Real-time weather data for informed decision-making
- 💰 **Market Rates**: Stay updated with current market prices
- 📘 **Khata Book**: Digital record keeping for farm operations
- 👤 **Profile Management**: Personalized user experience

## Tech Stack

- **Frontend**: React 18 with TypeScript
- **Styling**: TailwindCSS with Radix UI components
- **Build Tool**: Vite
- **State Management**: TanStack Query
- **Routing**: React Router
- **AI Integration**: Google Generative AI

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Abhrxdip/kisanai.git
cd kisanai
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Lint the codebase

## Deployment

This project is configured for deployment on Vercel. The frontend is automatically deployed when changes are pushed to the main branch.
